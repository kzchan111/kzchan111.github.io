论如何用hugo搭建个人博客

使用hugo搭建个人博客过程 安装hugo 打开hugo的官方网站:hugo官方网站,点击Quick Start快速进入，按照上面的步骤逐步进行，例如： 按照上面两张图片所示，依次复制里面的命令运行即可 完成教程中的所有步骤以后，新建一个文件：.gitignore，在.gitignore里面写上/public/，目的是使当前仓库不要储存public。 进入public：cd public，然后依次运行 git init git add .(注意空格与点) git commit -v 上传 在GitHub上新建一个仓库，仓库名必须是 GitHub用户名.github.io 创建好仓库后，依次运行下面的两条命令： 刷新GitHub，点击settings，找到GitHub Pages，选到master并保存，然后点击里面的网址就能看到个人的博客，个人博客搭建就完成了。
